### 기간 : 23.11.03 - 11.06

#### `분리한 컴포넌트`

#### TodoAdd

: form 태그의 input에 값을 넣고 등록하기를 클릭했을 때 working과 clear부분이 그려질 수 있는 위치를 만드는 컴포넌트

#### TodoCard

: 텍스트 입력 후 등록하기를 클릭했을 때 넣은 입력값이 출력되어 생성되는 카드를 붙이는 컴포넌트

#### OnToggle

: list의 isDone상태에 따라 카드가 붙는 위치와 버튼텍스트를 다르게 바꾸는 버튼 컴포넌트
